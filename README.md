# WORKSHOP ESTHETIQUE ET ALGORITHMIQUE

Ce dossier regroupe mes 5 projets réalisés durant cette semaine de workshop. Cette semaine a été une expérience très enrichissante. 

J'ai beaucoup aimé l'exercice qui m'a poussé à développer ma créativité au-delà du simple code.

C'était un super défi de réapprendre à utiliser Processing avec une approche différente de celle que j'avais déjà eue par le passé.

Globalement, je suis très satisfaite de mes 4 derniers travaux. Je trouve les avoir bien réussis et ils retranscrivent bien les émotions et les concepts que je voulais explorer.
