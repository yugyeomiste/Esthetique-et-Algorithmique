# Rendu : 14/01

Pour ce projet, j'ai voulu simuler la respiration humaine. J'ai utilisé le concept de fractales pour recréer la structure des bronches et des alvéoles.
Le mouvement de la souris contrôle l'air qui entre et sort des poumons.
