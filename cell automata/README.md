# Rendu : 13/01

Pour ce projet, j'ai voulu simuler la naissance d'une galaxie. J'ai utilisé le concept d'automates cellulaires pour créer une onde d'explosion qui part du centre et dépose les étoiles.

L'explosion commence par une explosion avec des couleurs chaudes (jaune, rouge) puis les couleurs changent progressivement vers le bleu et le violet pour représenterl'espace.

J'ai fais deux version une avec la grille pour bien voir les cellules automates et une sans.

# Instructions

Pour voir le projet, copier/coller le code dans Processing et lancer l'exécution.

BARRE ESPACE : Relancer le Big Bang (Réinitialiser l'univers).

# Sources

Pour la logique de base des cellules (règles de survie et de naissance), je me suis aidé du code "Game of Life" trouvé sur le site de Processing (voir sources https://processing.org/examples/gameoflife.html).
